# Vehicle Trajectory Prediction

딥러닝 기반 차량 궤적예측 과제를 정리하는 프로젝트 공간입니다.

## Problem

주변 차량의 과거 움직임을 바탕으로 이후 궤적을 예측하는 문제를 다룹니다. 자율주행 시스템에서 주변 객체의 미래 움직임을 예측하는 과정과 연결됩니다.

## Project Goal

- 차량 궤적예측 문제의 입력과 출력 구조를 정리합니다.
- RNN, attention, Transformer 계열 모델이 예측 문제에 어떻게 활용되는지 정리합니다.
- 구현 과정, 실험 조건, 결과, 한계를 기록합니다.
