# Problem Definition: DRT Policy Recommendation

## Problem

DRT 운행 정책과 관련된 텍스트 자료를 분석하고, 정책 제안으로 연결할 수 있는 근거를 구성하는 문제를 다룹니다.

이 프로젝트는 NLP/LLM을 단순히 사용하는 것보다, 텍스트 기반 분석 결과가 정책 의사결정에 어떻게 활용될 수 있는지 정리하는 데 초점을 둡니다.

## Input

```text
- DRT 운행 정책 관련 문서
- 정책 보고서 또는 공공 자료
- 관련 연구 논문
- 프로젝트에서 정리한 분석 기준
```

## Output

```text
- 정책 이슈 요약
- 분석 기준 또는 분류 체계
- 정책 제안 근거
- 프로젝트 보고서 또는 논문화 자료
```

## Key Questions

- 텍스트 자료에서 정책 제안에 필요한 근거를 어떻게 추출할 것인가?
- LLM 출력의 신뢰성을 어떻게 확인할 것인가?
- 정책 제안과 개인적 해석을 어떻게 구분해서 기록할 것인가?
- 프로젝트 결과물과 논문화 진행 기록을 어떻게 분리할 것인가?
