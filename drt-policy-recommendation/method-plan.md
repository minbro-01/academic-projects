# Method Plan: DRT Policy Recommendation

## Method Direction

DRT 관련 자료를 수집하고, NLP/LLM을 활용해 정책 제안에 필요한 논점과 근거를 정리합니다.

## Possible Steps

```text
1. 자료 수집 범위 정의
2. 문서 유형 분류
3. 핵심 정책 이슈 추출
4. NLP/LLM 기반 요약 또는 분류 실험
5. 결과 검토와 근거 정리
6. 정책 제안 문서화
```

## Analysis Units

| Unit | Description |
| --- | --- |
| Policy Issue | 정책 문서에서 반복적으로 등장하는 문제 또는 요구 |
| Evidence | 정책 제안을 뒷받침하는 자료, 통계, 문헌 근거 |
| Recommendation | 분석 결과를 바탕으로 한 제안 방향 |
| Limitation | 자료 부족, 모델 한계, 해석상 주의점 |

## Validation Notes

- LLM 출력은 원문 자료와 대조합니다.
- 정책 제안은 근거 자료와 함께 기록합니다.
- 불확실한 해석은 확정된 결론처럼 쓰지 않습니다.
